					
					Data Types
				----------------------------------------------
			There are 2 types of data types in Java:

Primitive - 	byte (1 byte) , short (2 byte) , int (4 byte) , long (8 byte), float (4 byte), double( 8 byte) , boolean (1 bit) and char (2 byte)

Non-Primitive - Strings, Arrays , Classes and Interface

		Primitive data types are predefined in java.                      Non Primitive are Defined by programmer (except for String)
		Cannot be used to call methods			    Can be used to call methods
		Always has a value				    Can be null
		starts with lower case				     starts with upper case
		size depends on data type			    all have the same size	
		

Integers types : byte, short, int and long
floating point types : float and double

byte : can store value between -128 To 127
short : can store numbers from -32768 to 32767
